Baseado em um formulário HTML que tem um campo de entrada do tipo (text) tendo o nome do campo: ‘frase’, o qual receberá do usuário uma frase. Escreva uma página com código de Linguagem PHP que recupere esta frase e ignore os espaçamentos para determinar se a frase é ou não um palíndromo. Considere que o usuário não digitará sinais de pontuação, entretanto, e letras em maiúsculas e letras minúsculas devem ser equalizadas. Frases palíndromos são frase de lidas de frente para trás e de trás para frente formam as mesmas frases.

Nessa questão iremos usar uma abordagem diferente. Vamos usar orientação a objeto.

Iniciaremos criando um arquivo com o nome de “palidromo.class.php”, dentro dele terá uma classe com o nome de “Palidromo", nessa classe teremos 5 métodos que serão responsáveis por realizar as ações necessárias, e é claro o nosso método construtor. São eles:

Agora vamos usar a função str_replace(), strtolower() e strrev() para construir nossos métodos. o método construtor vai receber um argumento, esse argumento é o nosso palíndromo.

Primeiro temos duas variáveis privadas com o nome de $palavra e $palavraOriginal, uma guardara a palavra já sem espaços e sem maiúsculas, mas em sua ordem original, já a outra guardara a palavra na ordem contraria.

Agora vamos para os métodos.

No método limparEspacos() é usado a função str_replace(), essa função leva três argumentos, sendo eles o valor que está sendo buscado, o valor que será usado para substituir o valor buscado, e a string que será afetada.

No método limparMaiusculo() usamos a função strtolower() para deixar tudo em minúsculas.

No método reverterString() usamos a função strrev() para reverter a string.
